# 🔐 Cyber Security Attacks Analysis

## 📌 Introduction

Cybersecurity has become a critical concern in today’s digital world due to the rapid increase in cyber attacks and data breaches. This project analyzes cybersecurity attack data to identify patterns, trends, and potential risks using data visualization techniques.

---

## 🎯 Objectives

* To analyze different types of cyber attacks
* To understand severity levels and associated risks
* To study the role of network protocols in attacks
* To examine anomaly scores for detecting unusual behavior
* To identify relationships between malware indicators and attack severity

---

## 🛠️ Tools & Technologies

* Python 
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 📊 Data Analysis & Visualizations

### 🔹 Top 10 Cyber Security Attacks

Shows the most frequent types of cyber attacks, helping identify the most common threats.

### 🔹 Severity Level Distribution

Displays how attacks are distributed across severity levels, indicating overall risk levels.

### 🔹 Network Protocol Usage in Attacks

Identifies which network protocols are most commonly targeted or exploited.

### 🔹 Source of Attack Logs

Highlights the systems or sources where attacks are detected.

### 🔹 Malware Indicators vs Severity Level

Shows the relationship between malware presence and attack severity.

### 🔹 Average Anomaly Score by Attack Type

Represents how unusual or suspicious each attack type is.

---

## 📈 Key Insights

* Certain attack types occur more frequently than others
* High severity attacks are often linked with malware presence
* Some network protocols are more vulnerable to attacks
* Anomaly scores help in detecting suspicious activities

---

## 🧾 Conclusion

This project highlights important patterns in cybersecurity attacks by analyzing various factors such as attack types, severity levels, protocols, and anomaly scores. The findings show that understanding these patterns can help in improving threat detection, strengthening security systems, and making better decisions to prevent cyber attacks.
